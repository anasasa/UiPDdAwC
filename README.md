## Workstation requirements

1. Recommended:
	- Ubuntu (Linux)
2. Installed:
	- azure-cli
3. Account on:
	- portal.azure.com
	
## Preparation of VM

Log in to Azure using command line:
```bash
az login --use-device-code
```

Download repository from github
```bash
git clone https://github.com/anasasa/UiPDdAwC
./UiPDdAwC/runme.sh
```
Set paramterer (index number) and run script
```bash
myindex=69303
./UiPDdAwC/runme.sh
```
