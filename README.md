## Workstation requirements

1. Recommended:
	- Ubuntu (Linux)
2. Installed:
	- azure-cli
3. Account on:
	- portal.azure.com
	
## Preparation of VM

Log in to Azure using command line:
```bash
az login --use-device-code
```

Download repository and run script file
```bash
git clone https://github.com/anasasa/UiPDdAwC
./runme.sh
```
